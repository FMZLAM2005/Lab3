# Exploratory Data Analysis on Student Performance Dataset

## Overview
This project performs Exploratory Data Analysis (EDA) on a student performance dataset using Python.

## Dataset Information
The dataset contains information about students, including:
- Gender
- Study Hours
- Attendance
- Previous Grade
- Final Grade

## Objectives
- Explore the dataset
- Analyze relationships between variables
- Visualize student performance patterns
- Identify factors affecting final grades

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn

## Analysis Performed
- Data inspection
- Missing value checking
- Statistical summary
- Distribution visualization
- Scatter plots analysis
- Correlation heatmap

## Conclusion
The analysis shows that study hours and attendance positively influence student final grades. Students with higher attendance and more study hours generally achieve better academic performance.
